# PhantomJS Cookie Value #

A utility using phantomjs to render a page and check the value of a cookie.

## Installation ##

* Clone this project
* Run ``npm install``

## Dependencies ##

* [PhantomJS](http://phantomjs.org/ "PhantomJS")

## Usage ##
``phantomjs cookieValue.js 1 http://www.mywebsite.com myCookieName``
